# 🚀 AI Business Strategy & Competitor Intelligence Agent

An AI-powered **Business Intelligence Agent** built using **Python**, **Streamlit**, **LangChain**, **Google Gemini**, and the **Groq API**.

The application helps businesses understand their competitive landscape by automatically researching competitor websites, extracting valuable insights, comparing market strategies, identifying business opportunities, and generating a professional business intelligence report with actionable recommendations.

Instead of spending hours manually analyzing competitors, users simply provide their business domain, target region, and (optionally) their own website. The AI agent performs the research, analyzes multiple competitors, and creates a complete strategic roadmap for business growth.

---

# 🌟 Overview

Competitive research is one of the most important yet time-consuming tasks for every business. Companies need to understand what their competitors offer, how they market their products, who their target audience is, and what opportunities exist in the market.

This project automates the entire process using AI agents.

The system researches competitor websites, extracts relevant business information, compares multiple competitors, identifies strengths and weaknesses, discovers market gaps, and recommends strategies that businesses can implement to improve their products, marketing, and overall growth.

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Business-Strategy-Agent.git

cd AI-Business-Strategy-Agent
```

---

## 2. Create a Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### macOS / Linux

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Configure API Keys

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key

GEMINI_API_KEY=your_gemini_api_key
```

---

## 5. Run the Application

```bash
streamlit run app.py
```

# 💡 How It Works

1. Enter your business domain.
2. Select the target region.
3. (Optional) Enter your own business website.
4. AI discovers relevant competitors.
5. The agent visits competitor websites.
6. Business information is extracted automatically.
7. AI analyzes competitors.
8. Market opportunities are identified.
9. Strategic recommendations are generated.
10. A professional business intelligence report is created.
